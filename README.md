# تطبيق إدارةvents وفعاليات مع إدارة دعوات وتذاكر
==========================

## ملخص المشروع
----------------

تطبيق إدارةvents وفعاليات مع إدارة دعوات وتذاكر هو تطبيق مفتوح المصدر يهدف إلى تسهيل إدارة الفعاليات والمناسبات، بما في ذلك إدارة الدعوات وتذاكر الحضور.

## هيكل المشروع
----------------

* `app/` : الملفات الخاصة بالتطبيق
 + `models/` : الملفات الخاصة بالنموذج
 + `controllers/` : الملفات الخاصة بالسيطرة
 + `routes/` : الملفات الخاصة بالطرق
* `database/` : الملفات الخاصة بالبيانات
 + `migrations/` : الملفات الخاصة بالتحديثات
 + `seeds/` : الملفات الخاصة بالزراعة
* `docker/` : الملفات الخاصة بالدوكر
 + `docker-compose.yml` : الملفات الخاصة بالدوكر
 + `Dockerfile` : الملفات الخاصة بالدوكر
* `tests/` : الملفات الخاصة بالاختبارات

## إعداد البيئة باستخدام دوكر
-----------------------------

### خطوات الإعداد

1. تأكد من توفر دوكر على جهازك.
2. افتح ترمينال وانتقل إلى مجلد المشروع.
3. استخدم الأمر التالي لتشغيل دوكر:
bash
docker-compose up

4. انتظر حتى يتم تشغيل البيئة.
5. افتح متصفح ويب ومرر إلى `http://localhost:8000` لفتح التطبيق.

## الوحدات والجداولات والدوائر
------------------------------

### الوحدات

* `events` : الوحدة الخاصة بالفعاليات
* `invitations` : الوحدة الخاصة بالدعوات
* `tickets` : الوحدة الخاصة بتذاكر الحضور

### الجداولات

* `events` : جدول الفعاليات
* `invitations` : جدول الدعوات
* `tickets` : جدول تذاكر الحضور

### الدوائر

* `admin` : الدائرة الخاصة بالمسؤولين
* `moderator` : الدائرة الخاصة بالمراقبين
* `user` : الدائرة الخاصة بالموظفين
