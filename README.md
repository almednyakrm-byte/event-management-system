# نظام إدارة أحداث وقاعات (الإقامة، الوجبات، الأنشطة)
==========================

## ملخص المشروع
----------------

نظام إدارة أحداث وقاعات (الإقامة، الوجبات، الأنشطة) هو تطبيق إلكتروني مصمم لتحسين إدارة الأحداث والقاعات في المؤسسات. يتيح للمستخدمين إدارة الإقامة والوجبات والأنشطة بشكل مريح وسهل.

## هيكل المشروع
----------------

### مكونات المشروع

* `backend`: يحتوي على الكود المصدر للباكند.
* `frontend`: يحتوي على الكود المصدر للفرونتند.
* `docker`: يحتوي على الملفات اللازمة لتشغيل المشروع باستخدام docker-compose.

### هيكل الملفات

* `backend`:
 + `app.py`: الملف الرئيسي للباكند.
 + `models.py`: يحتوي على تعريفات الجداول.
 + `routes.py`: يحتوي على تعريفات الطرق.
 + `schema.py`: يحتوي على تعريفات النماذج.
* `frontend`:
 + `index.html`: الصفحة الرئيسية للفرونتند.
 + `styles.css`: الملف الرئيسي للتصميم.
 + `script.js`: الملف الرئيسي للجافا سكريبت.
* `docker`:
 + `docker-compose.yml`: الملف الرئيسي لتشغيل المشروع باستخدام docker-compose.
 + `Dockerfile`: الملف الرئيسي لتشغيل الباكند باستخدام docker.

## تشغيل المشروع باستخدام docker-compose
--------------------------------------

### خطوات تشغيل المشروع

1. تأكد من أن docker-compose مثبت على جهازك.
2. انتقل إلى مجلد المشروع.
3. استخدم الأمر التالي لتشغيل المشروع:
bash
docker-compose up

4. انتظر حتى المشروع يبدأ تشغيله.
5. افتح متصفحك واكتب `http://localhost:5000` لفتح الصفحة الرئيسية للفرونتند.

## مكونات النظام
----------------

### الجداول

* `events`: يحتوي على جميع الأحداث.
* `rooms`: يحتوي على جميع القاعات.
* `accommodations`: يحتوي على جميع الإقامة.
* `meals`: يحتوي على جميع الوجبات.
* `activities`: يحتوي على جميع الأنشطة.

### الدور

* `admin`: يمكنه إدارة جميع الجداول.
* `moderator`: يمكنه إدارة الجداول باستثناء `events`.
* `user`: يمكنه إدارة الجداول باستثناء `events` و `rooms`.
